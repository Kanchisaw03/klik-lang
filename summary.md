# KLIK — Project Summary

## Overview

KLIK is a Rust-based multi-crate workspace implementing a programming language toolchain and ecosystem: compiler, runtime, standard library, formatter, linter, language server, package manager, and CLI tooling.

## High-level architecture

- **cli**: Command-line binary that invokes the compiler and tooling.
- **compiler/**: Core compiler crates (modular):
  - `ast`: AST types and span/position metadata.
  - `lexer`: Tokenizer (tokens and token utilities).
  - `parser`: Parser that builds the AST from tokens.
  - `semantic`: Semantic analysis and checks.
  - `type_system`: Type inference and checking.
  - `ir`: Intermediate representation.
  - `optimizer`: IR optimization passes.
  - `codegen`: Backend code generation (observed use of Cranelift).
  - `incremental`: Incremental compilation helpers.
- **formatter**: Pretty-printer that renders AST back to source.
- **linter**: Static analysis / lint rules.
- **lsp**: Language Server implementation for editor integration.
- **package_manager**: Package metadata and dependency handling.
- **runtime**: Runtime support (allocator, concurrency, error handling).
- **stdlib**: Standard library modules (collections, fs, io, math, net, strings, time).
- **docs/**: Architecture and language specification (Markdown).
- **examples/**: `.klik` example programs.
- **target/**: Build artifacts (compiled outputs) — not counted below.

Methodology: I counted all files under the workspace root except any path containing `target/` and excluded files with the `.md` extension, then summed line counts using a PowerShell script run from the project root. The script used (PowerShell):

```
$files = Get-ChildItem -Recurse -File -ErrorAction SilentlyContinue |
  Where-Object { $_.FullName -notmatch '\\target\\' -and $_.Extension -ne '.md' }
$sum = 0
foreach($f in $files){
  try{ $c = (Get-Content -Raw -ErrorAction Stop $f.FullName).Split("`n").Count } catch { $c = 0 }
  $sum += $c
}
Write-Output "FILES_COUNT:$($files.Count)"
Write-Output "LINES_COUNT:$sum"
```
